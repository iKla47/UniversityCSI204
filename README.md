## วิธีการใช้งาน

ผู้ใช้ต้องมีแอป (โปรแกรม) ดังต่อไปนี้ก่อน
- Visual Studio Code
- Node 22.22 หรือใหม่กว่า
- Docker ที่ทำงานแบบเนทีฟหรือผ่าน WSL

โคลนโปรเจคด้วยคำสั่ง `git clone` หลังจากนั้นเปิด Visual Studio Code แล้วเลือกเมนู Run > Debugging (หรือกด F5) เพื่อรันตัวโปรเจค
