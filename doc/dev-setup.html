วิธีการติดตั้ง ​

(เนื้อหานี้สันนิษฐานว่าคุณมีความรู้ในการใช้คอมพิวเตอร์ระดับกลาง-สูง และรู้วิธีใช้งานภาษา TypeScript)

ผู้ใช้ต้องมีแอป (โปรแกรม) ดังต่อไปนี้ก่อน

  • Git
  • Visual Studio Code
  • Node.js เวอร์ชั่น 22.22 หรือใหม่กว่า แนะนำให้ใช้งานเวอร์ชั่นหมวดหมู่ LTS เพื่อความเสถียรในการพัฒนาและการทดสอบระบบ
  • Docker Engine ที่สามารถใช้งานคอนเทนเนอร์ Linux ได้ซึ่งโดยปกติแล้วจะติดตั้งผ่าน Docker Desktop ถ้าคุณใช้ระบบวินโดว์ บน MacOS และ Linux คุณสามารถติดตั้งได้ผ่านทางตัวจัดการแพ็กเกจของระบบได้เลย (โปรดทราบตัวระบบเว็บต้องการแค่ Docker Engine เท่านั้นไม่ใช่ Docker Desktop)

เริ่มแรกให้โคลนตัวโครงงานจาก GitHub ด้วยคำสั่ง git clone (คุณสามารถดาวน์โหลดไฟล์ Zip แทนได้หากคุณไม่ต้องการใช้ Git)

BASH
git clone https://github.com/iKla47/UniversityCSI204

หลังจากนั้นเปิด Visual Studio Code แล้วเลือกเมนู Run > Debugging (หรือกด F5) เพื่อรันตัวโปรเจค

เคล็บลับ: คุณสามารถใช้คำสั่งนี้ได้ (ถ้าคุณไม่อยากใช้เมาส์)

BASH
cd UniversityCSI204
code .

ตัวระบบจะทำการติดตั้งชิ้นส่วนประกอบที่จำเป็น (Dependencies) ต่อการทำงานของระบบเว็บ ซึ่งกระบวนนี้อาจใช้เวลานานเมื่อทำงานครั้งแรก

INFO

ในตอนนี้ระบบยังไม่ทำการเรียก Docker Compose (Up) ให้โดยอัตโนมัติ คุณจำเป็นเรียกคำสั่งนั้นด้วยตนเอง
